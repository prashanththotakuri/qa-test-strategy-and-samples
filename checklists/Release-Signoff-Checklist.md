# Release Sign-Off Checklist

## Testing Completed
- [ ] Smoke completed
- [ ] Regression completed
- [ ] API checks completed
- [ ] Mobile checks completed (if applicable)

## Defects
- [ ] All critical defects closed
- [ ] High defects reviewed and accepted (if any)
- [ ] Defect summary shared with stakeholders

## Data & Environment
- [ ] Test data validated
- [ ] Environment stable
- [ ] Dependencies confirmed

## Approval
- [ ] QA sign-off
- [ ] Product/Business acceptance
- [ ] Release notes shared
